在绿联云NAS上安装Ubuntu虚拟机

发表于2026-03-21|更新于2026-03-21|教程
|总字数:512|阅读时长:1分钟|浏览量:|评论数:

前言

由于博主只有一台笔记本电脑，平时办公打游戏都是它，平时打游戏的时候，老是突然掉帧卡顿，在网上看到推荐降压超频，但是自己又不太懂，也不敢自己胡乱操作。

在咸鱼上花了40大洋给CPU降压了，玩儿了一晚上的游戏，确实帧数稳定了，很爽。

但是咸鱼的那位兄弟，把我的CPU虚拟化功能给关闭了，WSL2、VMWare直接就用不了了，我暂时也不想给它打开，毕竟关了这个确实能提升游戏性能。那我想本地有台Linux测试机器咋办，想了想，又得给家里的NAS上压力了。

目前的思路是，在NAS上装个Ubuntu的虚拟机，然后使用节点小宝来做P2P访问。

开始

下载镜像

首先，需要下载Ubuntu ISO镜像 -> Ubuntu镜像下载地址

创建虚拟机

按照官方文档的步骤来即可 -> 绿联NAS如何创建虚拟机？

安装Ubuntu

参考文章步骤 -> 安装 Ubuntu Server 24.04 LTS 图文教程

配置节点小宝

你需要到节点小宝官网，注册账号，然后在你的设备上，下载对应平台的客户端。 -> 节点小宝-下载页

登录进Ubuntu系统后，执行以下指令安装节点小宝

执行命令后，会在终端打印一个绑定链接，一个设备码

在节点小宝控制台，依次进入 设备管理 -> 添加远程设备 -> 使用设备码绑定，输入设备码进行绑定即可。

进入异地组网菜单，节点小宝异地组网，免费版仅支持同时开启2条组网，把虚拟机与本地打开即可。

复制组网IP，就可以像局域网的IP一样访问了。
