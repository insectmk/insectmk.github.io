我的世界老九门服务器说明

发表于2024-02-19|更新于2024-03-21|游戏
|字数总计:678|阅读时长:2分钟|阅读量:

客户端

下载JDK17，Windows x64 MSI Installer。

下载HMCL第三方启动器，游戏版本1.20.1，模组加载器为forge-47.2.20。

更新日志请查看我的世界老九门服务器日志。

必装Mod

  • 机械动力
  • GeckoLib Minecraft Java 版正运动学模型动画引擎
  • 雾中人恐怖模组
  • 穴居人恐怖模组
  • 拔刀剑2依赖-playeranimator
  • 拔刀剑2

推荐Mod

  • 自动钓鱼Forge版
  • JEI物品管理器
  • Xaero的世界地图
  • Xaero的小地图

联机

由于服务端模组过多，云服务负载过高，采用本机部署，服务器代理的形式部署。

支持内网穿透和EasyN2N虚拟局域网，内网穿透延迟较高，虚拟局域网延迟较低但需要下载EasyN2N/小黄鸭客户端。

需要服主本地运行，非24小时运行

  • 内网穿透

    直接访问insectmk.cn即可。

  • 虚拟局域网

    需下载EasyN2N/小黄鸭。

    导入链接：easyn2n://MTUwLjE1OC40Ni4xODg6OTUyN0BtaW5lY3JhZnRfb2xkY2xhc3NuaW5lQGF1dG9ATFVVTkNpMTRJREU9

模组教程

  • 【MC】拔刀剑2-刀的合成
  • Create机械动力 站内教程目录大纲

插件指令

deathteleport（死亡传送）

deathteleport

  • back：传送到最后死亡地点。
  • back：将选定的玩家传送到他们最后的死亡位置。
  • deathteleport：启用或禁用插件。
  • autotp：启用或禁用死亡后自动传送。相当于编辑配置文件中的 doAutoTeleportToDeathLocation 属性。不需要重新加载 插件。
  • autotp：更改自动传送到最后死亡位置之前的延迟。相当于编辑配置文件中的 autoTeleportToDeathLocationDelay 属性。不需要重新加载插件。
  • canceltp: 取消自动传送

SimpleWhitelist（白名单）

SimpleWhitelist

管理员可用

  • swl add/remove：add or remove player from whitelist
  • swl list：show list of whitelisted players
  • swl enable/disable：enable or disable plugin
  • swl reload：reload plugin

SkinsRestorer（皮肤）

SkinsRestorer

  • skin：主命令
  • skin：设置你的皮肤。
  • skin gui：查看服务器拥有的皮肤
  • skin url <skin.png url> [steve / slim]：从 .png url 设置皮肤
  • skin update：更新您当前的皮肤。
  • skin clear：清除你的皮肤。
  • skins：GUI

AdvancedTeleport（传送）

AdvancedTeleport

  • sethome：存储当前位置为一个传送地点
  • delhome：删除存储的某个传送地点
  • home：传送到存储的地点
  • homes：查看已存储的地点列表
  • at help：查看传送指令
  • tpa：传送到某人所在位置

服务器指令

  • gamemode <adventure|creative|spectator|survival>：游戏模式
  • difficulty <easy|hard|normal|peaceful>：服务器难度
  • mods：查看安装的模组
  • stop：关闭服务器
  • gamerule playersSleepingPercentage：设置百分之多少的人睡觉跳过黑夜
  • op：赋予某人管理员权限
  • deop：解除某人管理员权限
  • whitelist add：添加某个人到白名单
  • whitelist remove
  • whitelist list
  • whitelist off
  • whitelist on
  • whitelist reload
