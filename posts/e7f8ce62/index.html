ж·ұе…Ҙзұ»еҠ иҪҪжңәеҲ¶

еҲқиҜҶзұ»еҠ иҪҪиҝҮзЁӢ

дҪҝз”ЁжҹҗдёӘзұ»ж—¶пјҢеҰӮжһңиҜҘзұ»зҡ„classж–Үд»¶жІЎжңүеҠ иҪҪеҲ°еҶ…еӯҳж—¶пјҢеҲҷзі»з»ҹдјҡйҖҡиҝҮд»ҘдёӢдёүдёӘжӯҘйӘӨжқҘеҜ№иҜҘзұ»иҝӣиЎҢеҲқе§ӢеҢ–

1.зұ»зҡ„еҠ иҪҪпјҲLoadпјү вҶ’ 2.зұ»зҡ„иҝһжҺҘпјҲLinkпјү вҶ’ 3.зұ»зҡ„еҲқе§ӢеҢ–пјҲInitializeпјү

  • зұ»зҡ„еҠ иҪҪпјҲLoadпјүпјҡе°Ҷзұ»зҡ„classж–Үд»¶иҜ»е…ҘеҶ…еӯҳпјҢе№¶дёәд№ӢеҲӣе»әдёҖдёӘjava.lang.Classзҡ„еҜ№иұЎпјҢжӯӨиҝҮзЁӢз”ұзұ»еҠ иҪҪеҷЁпјҲClassLoader пјүе®ҢжҲҗ
  • зұ»зҡ„иҝһжҺҘпјҲLinkпјүпјҡе°Ҷзұ»дёӯзҡ„ж•°жҚ®еҠ иҪҪеҲ°еҗ„дёӘеҶ…еӯҳеҢәеҹҹдёӯ
  • зұ»зҡ„еҲқе§ӢеҢ–пјҲInitializeпјүпјҡJVMиҙҹиҙЈеҜ№зұ»иҝӣиЎҢеҲқе§ӢеҢ–

ж·ұе…Ҙзұ»еҠ иҪҪиҝҮзЁӢ

зұ»зҡ„е®Ңж•ҙз”ҹе‘Ҫе‘Ёжңҹ пјҡеҠ иҪҪгҖҒиҝһжҺҘпјҲйӘҢиҜҒгҖҒеҮҶеӨҮгҖҒи§ЈжһҗпјүгҖҒеҲқе§ӢеҢ–гҖҒдҪҝз”ЁгҖҒеҚёиҪҪ

  1. еҠ иҪҪ

    1. йҖҡиҝҮдёҖдёӘзұ»зҡ„е…Ёйҷҗе®ҡеҗҚжқҘиҺ·еҸ–е…¶е®ҡд№үзҡ„дәҢиҝӣеҲ¶еӯ—иҠӮжөҒ

    2. е°ҶиҝҷдёӘеӯ—иҠӮжөҒжүҖд»ЈиЎЁзҡ„зҡ„йқҷжҖҒеӯҳеӮЁз»“жһ„иҪ¬еҢ–дёәж–№жі•еҢәзҡ„иҝҗиЎҢж—¶ж•°жҚ®з»“жһ„

    3. еңЁе Ҷдёӯз”ҹжҲҗдёҖдёӘд»ЈиЎЁиҝҷдёӘзұ»зҡ„ClassеҜ№иұЎпјҢдҪңдёәж–№жі•еҢәдёӯиҝҷдәӣж•°жҚ®зҡ„и®ҝй—®е…ҘеҸЈ

    4. жіЁж„Ҹпјҡ зӣёеҜ№дәҺзұ»еҠ иҪҪиҝҮзЁӢзҡ„е…¶д»–йҳ¶ж®өиҖҢиЁҖпјҢеҠ иҪҪйҳ¶ж®өжҳҜеҸҜжҺ§жҖ§жңҖејәзҡ„йҳ¶ж®өпјҢеӣ дёәзЁӢеәҸе‘ҳеҸҜд»ҘдҪҝз”Ёзі»з»ҹзҡ„зұ»еҠ иҪҪеҷЁеҠ иҪҪпјҢиҝҳеҸҜд»ҘдҪҝз”ЁиҮӘе·ұзҡ„зұ»еҠ иҪҪеҷЁеҠ иҪҪпјҢеңЁиҝҷйҮҢжҲ‘д»¬еҸӘйңҖиҰҒзҹҘйҒ“зұ»еҠ иҪҪеҷЁзҡ„дҪңз”Ёе°ұжҳҜдёҠйқўиҷҡжӢҹжңәйңҖиҰҒе®ҢжҲҗзҡ„дёүд»¶дәӢ

  2. иҝһжҺҘ

    1. йӘҢиҜҒ

      1. ж–Үд»¶ж јејҸзҡ„йӘҢиҜҒпјҡйӘҢиҜҒ.classж–Үд»¶еӯ—иҠӮжөҒжҳҜеҗҰз¬ҰеҗҲclassж–Үд»¶зҡ„ж јејҸзҡ„и§„иҢғпјҢе№¶дё”иғҪеӨҹиў«еҪ“еүҚзүҲжң¬зҡ„иҷҡжӢҹжңәеӨ„зҗҶ

      2. е…ғж•°жҚ®йӘҢиҜҒпјҡдё»иҰҒжҳҜеҜ№еӯ—иҠӮз ҒжҸҸиҝ°зҡ„дҝЎжҒҜиҝӣиЎҢиҜӯд№үеҲҶжһҗпјҢд»ҘдҝқиҜҒе…¶жҸҸиҝ°зҡ„дҝЎжҒҜз¬ҰеҗҲjavaиҜӯиЁҖи§„иҢғзҡ„иҰҒжұӮпјҢжҜ”еҰӮиҜҙйӘҢиҜҒиҝҷдёӘзұ»жҳҜдёҚжҳҜжңүзҲ¶зұ»пјҢзұ»дёӯзҡ„еӯ—ж®өж–№жі•жҳҜдёҚжҳҜе’ҢзҲ¶зұ»еҶІзӘҒзӯүзӯүгҖӮ

      3. еӯ—иҠӮз ҒйӘҢиҜҒпјҡиҝҷжҳҜж•ҙдёӘйӘҢиҜҒиҝҮзЁӢжңҖеӨҚжқӮзҡ„йҳ¶ж®өпјҢдё»иҰҒжҳҜйҖҡиҝҮж•°жҚ®жөҒе’ҢжҺ§еҲ¶жөҒеҲҶжһҗпјҢзЎ®е®ҡзЁӢеәҸиҜӯд№үжҳҜеҗҲжі•зҡ„гҖҒз¬ҰеҗҲйҖ»иҫ‘зҡ„гҖӮеңЁе…ғж•°жҚ®йӘҢиҜҒйҳ¶ж®өеҜ№ж•°жҚ®зұ»еһӢеҒҡеҮәйӘҢиҜҒеҗҺпјҢиҝҷдёӘйҳ¶ж®өдё»иҰҒеҜ№зұ»зҡ„ж–№жі•еҒҡеҮәеҲҶжһҗпјҢдҝқиҜҒзұ»зҡ„ж–№жі•еңЁиҝҗиЎҢж—¶дёҚдјҡеҒҡеҮәеЁҒиғҒиҷҡжӢҹжңәе®үе…Ёзҡ„дәӢгҖӮ

      4. з¬ҰеҸ·еј•з”ЁйӘҢиҜҒпјҡе®ғжҳҜйӘҢиҜҒзҡ„жңҖеҗҺдёҖдёӘйҳ¶ж®өпјҢеҸ‘з”ҹеңЁиҷҡжӢҹжңәе°Ҷз¬ҰеҸ·еј•з”ЁиҪ¬еҢ–дёәзӣҙжҺҘеј•з”Ёзҡ„ж—¶еҖҷгҖӮдё»иҰҒжҳҜеҜ№зұ»иҮӘиә«д»ҘеӨ– пјҲеёёйҮҸжұ дёӯзҡ„еҗ„з§Қз¬ҰеҸ·еј•з”Ёпјү зҡ„дҝЎжҒҜиҝӣиЎҢж ЎйӘҢгҖӮзӣ®зҡ„жҳҜзЎ®дҝқи§ЈжһҗеҠЁдҪңиғҪеӨҹе®ҢжҲҗгҖӮ

      5. жіЁж„Ҹпјҡ еҜ№ж•ҙдёӘзұ»еҠ иҪҪжңәеҲ¶иҖҢиЁҖпјҢйӘҢиҜҒйҳ¶ж®өжҳҜдёҖдёӘеҫҲйҮҚиҰҒдҪҶжҳҜйқһеҝ…йңҖзҡ„йҳ¶ж®өпјҢеҰӮжһңжҲ‘д»¬зҡ„д»Јз ҒиғҪеӨҹзЎ®дҝқжІЎжңүй—®йўҳпјҢйӮЈд№ҲжҲ‘д»¬е°ұжІЎжңүеҝ…иҰҒеҺ»йӘҢиҜҒпјҢжҜ•з«ҹйӘҢиҜҒйңҖиҰҒиҠұиҙ№дёҖе®ҡзҡ„зҡ„ж—¶й—ҙгҖӮеҪ“з„¶жҲ‘д»¬еҸҜд»ҘдҪҝз”Ё

        -Xverfty:noneжқҘе…ій—ӯеӨ§йғЁеҲҶзҡ„йӘҢиҜҒгҖӮ

    2. еҮҶеӨҮ - йҮҚиҰҒ

      еҮҶеӨҮйҳ¶ж®өдё»иҰҒдёәзұ»еҸҳйҮҸпјҲstaticпјүеҲҶй…ҚеҶ…еӯҳе№¶и®ҫзҪ®еҲқе§ӢеҖјгҖӮиҝҷдәӣеҶ…еӯҳйғҪеңЁж–№жі•еҢәеҲҶй…ҚгҖӮеңЁиҝҷдёӘйҳ¶ж®өжҲ‘д»¬еҸӘйңҖиҰҒжіЁж„ҸдёӨзӮ№е°ұеҘҪдәҶпјҢзұ»еҸҳйҮҸе’ҢеҲқе§ӢеҖјдёӨдёӘе…ій”®иҜҚпјҡ

      1. зұ»еҸҳйҮҸпјҲstaticпјүпјҡдјҡеҲҶй…ҚеҶ…еӯҳпјҢдҪҶдёҚдјҡеҜ№еә”зҡ„еҲҶй…ҚеҖјпјҢе…¶ж¬Ўе®һдҫӢеҸҳйҮҸдёҚдјҡеҲҶй…Қз©әй—ҙпјҢеӣ дёәе®һдҫӢеҸҳйҮҸдё»иҰҒйҡҸзқҖеҜ№иұЎзҡ„е®һдҫӢеҢ–дёҖеқ—еҲҶй…ҚеҲ°javaе ҶеҶ…еӯҳдёӯ

      2. еҲқе§ӢеҖјпјҡиҝҷйҮҢзҡ„еҲқе§ӢеҖјжҢҮзҡ„жҳҜж•°жҚ®зұ»еһӢй»ҳи®ӨеҖјпјҢиҖҢдёҚжҳҜд»Јз Ғдёӯиў«жҳҫзӨәиөӢдәҲзҡ„еҖј

      жҜ”еҰӮ1пјҡpublic static int value = 1;

      еңЁиҝҷйҮҢеҮҶеӨҮйҳ¶ж®өиҝҮеҗҺзҡ„valueеҖјдёә0пјҢиҖҢдёҚжҳҜ1иөӢеҖјдёә1зҡ„еҠЁдҪңеңЁеҲқе§ӢеҢ–йҳ¶ж®ө

      жҜ”еҰӮ2пјҡpublic static final int value = 1;

      еҗҢж—¶иў«finalе’Ңstaticдҝ®йҘ°еҮҶеӨҮйҳ¶ж®өд№ӢеҗҺе°ұжҳҜ1дәҶпјҢеӣ дёәstatic finalеңЁзј–иҜ‘еҷЁе°ұе°Ҷз»“жһңж”ҫе…Ҙи°ғз”Ёе®ғзҡ„зұ»зҡ„еёёйҮҸжұ дёӯ

    3. и§Јжһҗ

      и§Јжһҗйҳ¶ж®өдё»иҰҒжҳҜиҷҡжӢҹжңәе°ҶеёёйҮҸжұ дёӯзҡ„з¬ҰеҸ·еј•з”ЁиҪ¬еҢ–дёәзӣҙжҺҘеј•з”Ёзҡ„иҝҮзЁӢ

      1. з¬ҰеҸ·еј•з”Ёпјҡд»ҘдёҖз»„з¬ҰеҸ·жқҘжҸҸиҝ°жүҖеј•з”Ёзҡ„зӣ®ж ҮпјҢеҸҜд»ҘжҳҜд»»дҪ•еҪўејҸзҡ„еӯ—йқўйҮҸпјҢеҸӘиҰҒжҳҜиғҪж— жӯ§д№үзҡ„е®ҡдҪҚеҲ°зӣ®ж Үе°ұеҘҪпјҢе°ұеҘҪжҜ”еңЁзҸӯзә§дёӯпјҢиҖҒеёҲеҸҜд»Ҙз”Ёеј дёүжқҘд»ЈиЎЁдҪ пјҢд№ҹеҸҜд»Ҙз”ЁдҪ зҡ„еӯҰеҸ·жқҘд»ЈиЎЁдҪ пјҢдҪҶж— и®әд»»дҪ•ж–№ејҸиҝҷдәӣйғҪеҸӘжҳҜдёҖдёӘд»ЈеҸ·пјҲз¬ҰеҸ·пјүпјҢиҝҷдёӘд»ЈеҸ·жҢҮеҗ‘дҪ пјҲз¬ҰеҸ·еј•з”Ёпјү
      2. зӣҙжҺҘеј•з”ЁпјҡзӣҙжҺҘеј•з”ЁжҳҜеҸҜд»ҘжҢҮеҗ‘зӣ®ж Үзҡ„жҢҮй’ҲгҖҒзӣёеҜ№еҒҸз§»йҮҸжҲ–иҖ…жҳҜдёҖдёӘиғҪзӣҙжҺҘжҲ–й—ҙжҺҘе®ҡдҪҚеҲ°зӣ®ж Үзҡ„еҸҘжҹ„гҖӮе’ҢиҷҡжӢҹжңәе®һзҺ°зҡ„еҶ…еӯҳжңүе…іпјҢдёҚеҗҢзҡ„иҷҡжӢҹжңәзӣҙжҺҘеј•з”ЁдёҖиҲ¬дёҚеҗҢ
      3. иЎҘе……пјҡ и§ЈжһҗеҠЁдҪңдё»иҰҒй’ҲеҜ№зұ»жҲ–жҺҘеҸЈгҖҒеӯ—ж®өгҖҒзұ»ж–№жі•гҖҒжҺҘеҸЈж–№жі•гҖҒж–№жі•зұ»еһӢгҖҒж–№жі•еҸҘжҹ„е’Ңи°ғз”ЁзӮ№йҷҗе®ҡз¬Ұ7зұ»з¬ҰеҸ·еј•з”ЁиҝӣиЎҢ

    4. еҲқе§ӢеҢ–

      иҝҷжҳҜзұ»еҠ иҪҪжңәеҲ¶зҡ„жңҖеҗҺдёҖжӯҘпјҢеңЁиҝҷдёӘйҳ¶ж®өпјҢjavaзЁӢеәҸд»Јз ҒжүҚејҖе§ӢзңҹжӯЈжү§иЎҢгҖӮеңЁеҮҶеӨҮйҳ¶ж®өе·Із»Ҹдёәзұ»еҸҳйҮҸиөӢиҝҮдёҖж¬ЎеҖјгҖӮеңЁеҲқе§ӢеҢ–йҳ¶з«ҜпјҢзЁӢеәҸе‘ҳеҸҜд»Ҙж №жҚ®иҮӘе·ұзҡ„йңҖжұӮжқҘиөӢеҖјдәҶгҖӮдёҖеҸҘиҜқжҸҸиҝ°иҝҷдёӘйҳ¶ж®өе°ұжҳҜжү§иЎҢзұ»жһ„йҖ еҷЁclinit()ж–№жі•зҡ„иҝҮзЁӢгҖӮ

      еңЁеҲқе§ӢеҢ–йҳ¶ж®өпјҢдё»иҰҒдёәзұ»зҡ„йқҷжҖҒпјҲstiticпјүеҸҳйҮҸиөӢдәҲжӯЈзЎ®зҡ„еҲқе§ӢеҖјпјҢJVMиҙҹиҙЈеҜ№зұ»иҝӣиЎҢеҲқе§ӢеҢ–пјҢдё»иҰҒеҜ№зұ»еҸҳйҮҸпјҲstiticпјүиҝӣиЎҢеҲқе§ӢеҢ–гҖӮеңЁJavaдёӯеҜ№зұ»еҸҳйҮҸиҝӣиЎҢеҲқе§ӢеҖји®ҫе®ҡжңүдёӨз§Қж–№ејҸпјҡ

      1. еЈ°жҳҺзұ»еҸҳйҮҸжҳҜжҢҮе®ҡеҲқе§ӢеҖј
      2. дҪҝз”ЁйқҷжҖҒд»Јз Ғеқ—дёәзұ»еҸҳйҮҸжҢҮе®ҡеҲқе§ӢеҖј

      иЎҘе……пјҡclinit() ж–№жі•е…·жңүд»ҘдёӢзү№зӮ№пјҡ

      1. з”ұзј–иҜ‘еҷЁиҮӘеҠЁж”¶йӣҶзұ»дёӯжүҖжңүзұ»еҸҳйҮҸпјҲstaticпјүзҡ„иөӢеҖјеҠЁдҪңе’ҢйқҷжҖҒиҜӯеҸҘеқ—пјҲstatic{} еқ—пјүдёӯзҡ„иҜӯеҸҘеҗҲе№¶дә§з”ҹзҡ„пјҢзј–иҜ‘еҷЁж”¶йӣҶзҡ„йЎәеәҸз”ұиҜӯеҸҘеңЁжәҗж–Үд»¶дёӯеҮәзҺ°зҡ„йЎәеәҸеҶіе®ҡгҖӮзү№еҲ«жіЁж„Ҹзҡ„жҳҜпјҢйқҷжҖҒиҜӯеҸҘеқ—еҸӘиғҪи®ҝй—®еҲ°е®ҡд№үеңЁе®ғд№ӢеүҚзҡ„зұ»еҸҳйҮҸпјҢе®ҡд№үеңЁе®ғд№ӢеҗҺзҡ„зұ»еҸҳйҮҸеҸӘиғҪиөӢеҖјпјҢдёҚиғҪи®ҝй—®гҖӮдҫӢеҰӮд»ҘдёӢд»Јз Ғ

      2. дёҺзұ»зҡ„жһ„йҖ еҮҪж•°пјҲжҲ–иҖ…иҜҙе®һдҫӢжһ„йҖ еҷЁ init()пјүдёҚеҗҢпјҢдёҚйңҖиҰҒжҳҫејҸзҡ„и°ғз”ЁзҲ¶зұ»зҡ„жһ„йҖ еҷЁгҖӮиҷҡжӢҹжңәдјҡиҮӘеҠЁдҝқиҜҒеңЁеӯҗзұ»зҡ„ clinit() ж–№жі•иҝҗиЎҢд№ӢеүҚпјҢзҲ¶зұ»зҡ„ clinit() ж–№жі•е·Із»Ҹжү§иЎҢз»“жқҹгҖӮеӣ жӯӨиҷҡжӢҹжңәдёӯз¬¬дёҖдёӘжү§иЎҢ clinit() ж–№жі•зҡ„зұ»иӮҜе®ҡдёә java.lang.ObjectгҖӮз”ұдәҺзҲ¶зұ»зҡ„ clinit() ж–№жі•е…Ҳжү§иЎҢпјҢд№ҹе°ұж„Ҹе‘ізқҖзҲ¶зұ»дёӯе®ҡд№үзҡ„йқҷжҖҒиҜӯеҸҘеқ—иҰҒдјҳдәҺеӯҗзұ»зҡ„еҸҳйҮҸиөӢеҖјж“ҚдҪңгҖӮдҫӢеҰӮд»ҘдёӢд»Јз Ғпјҡ

      5. иҷҡжӢҹжңәдјҡдҝқиҜҒдёҖдёӘзұ»зҡ„ clinit() ж–№жі•еңЁеӨҡзәҝзЁӢзҺҜеўғдёӢиў«жӯЈзЎ®зҡ„еҠ й”Ғе’ҢеҗҢжӯҘпјҢеҰӮжһңеӨҡдёӘзәҝзЁӢеҗҢж—¶еҲқе§ӢеҢ–дёҖдёӘзұ»пјҢеҸӘдјҡжңүдёҖдёӘзәҝзЁӢжү§иЎҢиҝҷдёӘзұ»зҡ„ clinit() ж–№жі•пјҢе…¶е®ғзәҝзЁӢйғҪдјҡйҳ»еЎһзӯүеҫ…пјҢзӣҙеҲ°жҙ»еҠЁзәҝзЁӢжү§иЎҢ clinit() ж–№жі•е®ҢжҜ•гҖӮеҰӮжһңеңЁдёҖдёӘзұ»зҡ„ clinit() ж–№жі•дёӯжңүиҖ—ж—¶зҡ„ж“ҚдҪңпјҢе°ұеҸҜиғҪйҖ жҲҗеӨҡдёӘзәҝзЁӢйҳ»еЎһпјҢеңЁе®һйҷ…иҝҮзЁӢдёӯжӯӨз§Қйҳ»еЎһеҫҲйҡҗи”ҪгҖӮ

      6. JVMеҲқе§ӢеҢ–жӯҘйӘӨпјҡ

        1. еҒҮеҰӮиҝҷдёӘзұ»иҝҳжІЎжңүиў«еҠ иҪҪе’ҢиҝһжҺҘпјҢеҲҷзЁӢеәҸе…ҲеҠ иҪҪе№¶иҝһжҺҘиҜҘзұ»
2. еҒҮеҰӮиҜҘзұ»зҡ„зӣҙжҺҘзҲ¶зұ»иҝҳжІЎжңүиў«еҲқе§ӢеҢ–пјҢеҲҷе…ҲеҲқе§ӢеҢ–е…¶зӣҙжҺҘзҲ¶зұ»
3. еҒҮеҰӮзұ»дёӯжңүеҲқе§ӢеҢ–иҜӯеҸҘпјҢеҲҷзі»з»ҹдҫқж¬Ўжү§иЎҢиҝҷдәӣеҲқе§ӢеҢ–иҜӯеҸҘ

      7. зұ»еҲқе§ӢеҢ–ж—¶жңәпјҡ

        еҸӘжңүеҪ“еҜ№зұ»зҡ„дё»еҠЁдҪҝз”Ёзҡ„ж—¶еҖҷжүҚдјҡеҜјиҮҙзұ»зҡ„еҲқе§ӢеҢ–пјҢзұ»зҡ„дё»еҠЁдҪҝз”ЁеҢ…жӢ¬д»ҘдёӢе…ӯз§Қпјҡ

        1. еҲӣе»әзұ»зҡ„е®һдҫӢпјҢд№ҹе°ұжҳҜnewзҡ„ж–№ејҸ

        2. и®ҝй—®жҹҗдёӘзұ»жҲ–жҺҘеҸЈзҡ„йқҷжҖҒеҸҳйҮҸпјҢжҲ–иҖ…еҜ№иҜҘйқҷжҖҒеҸҳйҮҸиөӢеҖј

        3. и°ғз”Ёзұ»зҡ„йқҷжҖҒж–№жі•

        4. еҸҚе°„

        5. еҲқе§ӢеҢ–жҹҗдёӘзұ»зҡ„еӯҗзұ»пјҢеҲҷе…¶зҲ¶зұ»д№ҹдјҡиў«еҲқе§ӢеҢ–

        6. JavaиҷҡжӢҹжңәеҗҜеҠЁж—¶иў«ж ҮжҳҺдёәеҗҜеҠЁзұ»зҡ„зұ»пјҢзӣҙжҺҘдҪҝз”Ё java.exeе‘Ҫд»ӨжқҘиҝҗиЎҢжҹҗдёӘдё»зұ»

еҲ©з”Ёзұ»еҠ иҪҪиҝҮзЁӢзҗҶи§ЈйқўиҜ•йўҳ

зұ»еҠ иҪҪеҷЁ

зұ»еҠ иҪҪеҷЁеҲҶзұ»

д»Һ Java иҷҡжӢҹжңәзҡ„и§’еәҰжқҘи®ІпјҢеҸӘеӯҳеңЁд»ҘдёӢдёӨз§ҚдёҚеҗҢзҡ„зұ»еҠ иҪҪеҷЁпјҡ

  1. еҗҜеҠЁзұ»еҠ иҪҪеҷЁпјҲBootstrap ClassLoaderпјүпјҡиҝҷдёӘзұ»еҠ иҪҪеҷЁз”Ё C++ е®һзҺ°пјҢжҳҜиҷҡжӢҹжңәиҮӘиә«зҡ„дёҖйғЁеҲҶ
  2. жүҖжңүе…¶д»–зұ»зҡ„еҠ иҪҪеҷЁпјҢиҝҷдәӣзұ»з”ұ Java е®һзҺ°пјҢзӢ¬з«ӢдәҺиҷҡжӢҹжңәеӨ–йғЁпјҢе№¶дё”е…ЁйғҪз»§жүҝиҮӘжҠҪиұЎзұ» java.lang.ClassLoader

д»Һ Java ејҖеҸ‘дәәе‘ҳзҡ„и§’еәҰзңӢпјҢзұ»еҠ иҪҪеҷЁеҸҜд»ҘеҲ’еҲҶеҫ—жӣҙз»ҶиҮҙдёҖдәӣпјҡ

  1. еҗҜеҠЁзұ»еҠ иҪҪеҷЁпјҲBootstrap ClassLoaderпјүпјҡ жңҖйЎ¶еұӮзҡ„зұ»еҠ иҪҪеҷЁпјҢиҙҹиҙЈеҠ иҪҪ JAVA_HOME\lib зӣ®еҪ•дёӯзҡ„пјҢжҲ–йҖҡиҝҮ-XbootclasspathеҸӮж•°жҢҮе®ҡи·Ҝеҫ„дёӯзҡ„пјҢдё”иў«иҷҡжӢҹжңәи®ӨеҸҜпјҲжҢүж–Үд»¶еҗҚиҜҶеҲ«пјҢеҰӮrt.jarпјүзҡ„зұ»гҖӮ
  2. жү©еұ•зұ»еҠ иҪҪеҷЁпјҲExtension ClassLoaderпјүиҝҷдёӘзұ»еҠ иҪҪеҷЁжҳҜз”ұ ExtClassLoaderпјҲsun.misc.Launcher$ExtClassLoaderпјүе®һзҺ°зҡ„гҖӮ иҙҹиҙЈеҠ иҪҪ JAVA_HOME\lib\ext зӣ®еҪ•дёӯзҡ„пјҢжҲ–йҖҡиҝҮjava.ext.dirsзі»з»ҹеҸҳйҮҸжҢҮе®ҡи·Ҝеҫ„дёӯзҡ„зұ»еә“
  3. еә”з”ЁзЁӢеәҸзұ»еҠ иҪҪеҷЁпјҲApplication ClassLoaderпјүиҝҷдёӘзұ»еҠ иҪҪеҷЁжҳҜз”ұ AppClassLoaderпјҲsun.misc.Launcher$AppClassLoaderпјүе®һзҺ°зҡ„гҖӮ д№ҹеҸ«еҒҡзі»з»ҹзұ»еҠ иҪҪеҷЁпјҢеҸҜд»ҘйҖҡиҝҮgetSystemClassLoader()иҺ·еҸ–пјҢиҙҹиҙЈеҠ иҪҪз”ЁжҲ·и·Ҝеҫ„пјҲclasspathпјүдёҠзҡ„зұ»еә“гҖӮеҰӮжһңжІЎжңүиҮӘе®ҡд№үзұ»еҠ иҪҪеҷЁпјҢдёҖиҲ¬иҝҷдёӘе°ұжҳҜй»ҳи®Өзҡ„зұ»еҠ иҪҪеҷЁгҖӮ

зұ»еҠ иҪҪеҷЁд№Ӣй—ҙзҡ„еұӮж¬Ўе…ізі»

еҸҢдәІе§”жҙҫеҺҹеҲҷ

  1. еҸҜд»ҘйҒҝе…ҚйҮҚеӨҚеҠ иҪҪпјҢзҲ¶зұ»е·Із»ҸеҠ иҪҪдәҶпјҢеӯҗзұ»е°ұдёҚйңҖиҰҒеҶҚж¬ЎеҠ иҪҪ
  2. жӣҙеҠ е®үе…ЁпјҢеҫҲеҘҪзҡ„и§ЈеҶідәҶеҗ„дёӘзұ»еҠ иҪҪеҷЁзҡ„еҹәзЎҖзұ»зҡ„з»ҹдёҖй—®йўҳпјҢеҰӮжһңдёҚдҪҝз”ЁиҜҘз§Қж–№ејҸпјҢйӮЈд№Ҳз”ЁжҲ·еҸҜд»ҘйҡҸж„Ҹе®ҡд№үзұ»еҠ иҪҪеҷЁжқҘеҠ иҪҪж ёеҝғapiпјҢдјҡеёҰжқҘзӣёе…ійҡҗжӮЈгҖӮ

еҸҢдәІе§”жҙҫжЁЎеһӢзҡ„д»Јз Ғе®һзҺ°

еҸҢдәІе§”жҙҫжЁЎеһӢзҡ„д»Јз Ғе®һзҺ°йӣҶдёӯеңЁjava.lang.ClassLoaderзҡ„loadClass()ж–№жі•еҪ“дёӯ

1. жЈҖжҹҘзұ»жҳҜеҗҰиў«еҠ иҪҪпјҢжІЎжңүеҲҷи°ғз”ЁзҲ¶зұ»еҠ иҪҪеҷЁзҡ„loadClass()ж–№жі•пјӣ
2. иӢҘзҲ¶зұ»еҠ иҪҪеҷЁдёәз©әпјҢеҲҷй»ҳи®ӨдҪҝз”ЁеҗҜеҠЁзұ»еҠ иҪҪеҷЁдҪңдёәзҲ¶еҠ иҪҪеҷЁпјӣ
3. иӢҘзҲ¶зұ»еҠ иҪҪеӨұиҙҘпјҢжҠӣеҮәClassNotFoundException ејӮеёёеҗҺпјҢеҶҚи°ғз”ЁиҮӘе·ұзҡ„findClass() ж–№жі•гҖӮ

иҮӘе®ҡд№үзұ»еҠ иҪҪеҷЁ

йҰ–е…ҲпјҢжҲ‘д»¬е®ҡд№үдёҖдёӘеҫ…еҠ иҪҪзҡ„жҷ®йҖҡJavaзұ»:Test.javaгҖӮж”ҫеңЁcom.dream.testеҢ…дёӢ:

жіЁж„Ҹпјҡ

еҰӮжһңдҪ жҳҜзӣҙжҺҘеңЁеҪ“еүҚйЎ№зӣ®йҮҢйқўеҲӣе»әпјҢеҫ…Test.javaзј–иҜ‘еҗҺпјҢиҜ·жҠҠTest.classж–Үд»¶жӢ·иҙқиө°пјҢеҶҚе°ҶTest.java`еҲ йҷӨгҖӮеӣ дёәеҰӮжһңTest.classеӯҳж”ҫеңЁеҪ“еүҚйЎ№зӣ®дёӯпјҢж №жҚ®еҸҢдәІе§”жҙҫжЁЎеһӢеҸҜзҹҘпјҢдјҡйҖҡиҝҮsun.misc.Launcher$AppClassLoaderзұ»еҠ иҪҪеҷЁеҠ иҪҪгҖӮдёәдәҶи®©жҲ‘д»¬иҮӘе®ҡд№үзҡ„зұ»еҠ иҪҪеҷЁеҠ иҪҪпјҢжҲ‘д»¬жҠҠTest.classж–Үд»¶ж”ҫе…ҘеҲ°е…¶д»–зӣ®еҪ•гҖӮ

еңЁжң¬дҫӢдёӯпјҢжҲ‘д»¬Person.classж–Үд»¶еӯҳж”ҫзҡ„зӣ®еҪ•еҰӮдёӢпјҡ C:\code\com\dream\test\Person.class

жҺҘдёӢжқҘе°ұжҳҜиҮӘе®ҡд№үжҲ‘д»¬зҡ„зұ»еҠ иҪҪеҷЁ пјҡ

иҮӘе®ҡд№үзұ»еҠ иҪҪеҷЁзҡ„еә”з”ЁеңәжҷҜ

еј•е…ҘпјҡTomcatе®№еҷЁпјҢжҜҸдёӘWebAppжңүиҮӘе·ұзҡ„ClassLoader,еҠ иҪҪжҜҸдёӘWebAppзҡ„ClassPathи·Ҝеҫ„дёҠзҡ„зұ»пјҢдёҖж—ҰйҒҮеҲ°TomcatиҮӘеёҰзҡ„JarеҢ…е°ұе§”жүҳз»ҷCommonClassLoaderеҠ иҪҪ

  1. еҠ еҜҶпјҡJavaд»Јз ҒеҸҜд»ҘиҪ»жҳ“зҡ„иў«еҸҚзј–иҜ‘пјҢеҰӮжһңдҪ йңҖиҰҒжҠҠиҮӘе·ұзҡ„д»Јз ҒиҝӣиЎҢеҠ еҜҶд»ҘйҳІжӯўеҸҚзј–иҜ‘пјҢеҸҜд»Ҙе…Ҳе°Ҷзј–иҜ‘еҗҺзҡ„д»Јз Ғз”Ёжҹҗз§ҚеҠ еҜҶз®—жі•еҠ еҜҶпјҢзұ»еҠ еҜҶеҗҺе°ұдёҚиғҪеҶҚз”ЁJavaзҡ„ClassLoaderеҺ»еҠ иҪҪзұ»дәҶпјҢиҝҷж—¶е°ұйңҖиҰҒиҮӘе®ҡд№үClassLoaderеңЁеҠ иҪҪзұ»зҡ„ж—¶еҖҷе…Ҳи§ЈеҜҶзұ»пјҢз„¶еҗҺеҶҚеҠ иҪҪгҖӮ
  2. д»Һйқһж ҮеҮҶзҡ„жқҘжәҗеҠ иҪҪд»Јз ҒпјҡеҰӮжһңдҪ зҡ„еӯ—иҠӮз ҒжҳҜж”ҫеңЁж•°жҚ®еә“гҖҒз”ҡиҮіжҳҜеңЁдә‘з«ҜпјҢе°ұеҸҜд»ҘиҮӘе®ҡд№үзұ»еҠ иҪҪеҷЁпјҢд»ҺжҢҮе®ҡзҡ„жқҘжәҗеҠ иҪҪзұ»гҖӮ

д»ҘдёҠдёӨз§Қжғ…еҶөеңЁе®һйҷ…дёӯзҡ„з»јеҗҲиҝҗз”ЁпјҡжҜ”еҰӮдҪ зҡ„еә”з”ЁйңҖиҰҒйҖҡиҝҮзҪ‘з»ңжқҘдј иҫ“ Java зұ»зҡ„еӯ—иҠӮз ҒпјҢдёәдәҶе®үе…ЁжҖ§пјҢиҝҷдәӣеӯ—иҠӮз Ғз»ҸиҝҮдәҶеҠ еҜҶеӨ„зҗҶгҖӮиҝҷдёӘж—¶еҖҷдҪ е°ұйңҖиҰҒиҮӘе®ҡд№үзұ»еҠ иҪҪеҷЁжқҘд»ҺжҹҗдёӘзҪ‘з»ңең°еқҖдёҠиҜ»еҸ–еҠ еҜҶеҗҺзҡ„еӯ—иҠӮд»Јз ҒпјҢжҺҘзқҖиҝӣиЎҢи§ЈеҜҶе’ҢйӘҢиҜҒпјҢжңҖеҗҺе®ҡд№үеҮәеңЁJavaиҷҡжӢҹжңәдёӯиҝҗиЎҢзҡ„зұ»гҖӮ
