[2024-06-06]-Y7000P2022-更新一些软件

git

当初Git是绿色安装的（解压再加个环境变量）。

现在更新只需要重新到Git下载页下载64-bit Git for Windows Portable版本的压缩包（双击直接解压）。

然后覆盖之前的Git文件夹就行了，可以CMD输入个git -v看下是否更新，目前Windows最新版本是2.45.2.windows.1。

VMware Workstation Pro

VMware Workstation Pro 17向个人用户免费开放了，不用再找激活码了，升级一下。

到BROADCOM ProductDownloads下载个人版本就可以了。

注册时千万千万不要用中文，目前网站好像还不支持中文编码。

SwitchHosts

一个管理、切换多个hosts方案的工具，并且一个免费开源，对开发人员来说简直不要太好用。

SwitchHosts官网

下载软件包

Windows可以将快捷方式设置为管理员运行（右键 -> 属性 -> 快捷方式 -> 高级 -> 用管理员身份运行），下次直接双击快捷键就可以了，不用每次右键管理员运行。

显卡驱动

最近Chrome全屏老是闪屏，估计是显卡驱动的问题，把图形加速关了就行了，但是网页帧率又不太稳定，更新显卡驱动到最新版本GeForce Game Ready:555.99试试。
