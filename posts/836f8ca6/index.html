Day01

学习方法

  1. 每天知识点的梳理，形成每天的知识点文档
  2. 每天老师代码编写至少3遍，每一遍都要写注意事项
  3. 作业（锻炼思维能力，第二天老师会去评讲）
  4. 复习（每天复习前一天的内容，每周天复习本周的内容）
  5. 预习（培养自学能力）
  6. 与老师交流，看哪些方面能深入

一、软件开发基础

1.什么是JavaEE

接收到客户端的请求，并通过响应返回给客户端数据的技术

2.常有的开发语言

C/C++：嵌入式

HTML、CSS、JavaScript、JQuery、VUE、BootStrap：网页

Java：移动端、服务器、数据库

Python：数据分析

二、Java开发环境搭建

1.什么是JDK和JRE

目的：做菜

锅碗瓢盆、菜刀、菜板：工具

小煎鸡脚：食材

厨房：环境

目的：写代码

JDK：Java开发工具包

Java：代码

JRE：Java运行时环境

2.安装JDK

注意：

  1. 如果你仅仅运行java程序，只用安装JRE即可，但是你要编写Java程序，就必须安装JDK和JRE，所以JDK安装中包含了JRE

    1. 安装路径建议用默认路径

  2. 如果使用自定义路径，路径中不要使用中文

  3. 安装后不会像QQ有开启软件的图标

  4. 如果你安装了JDK和JRE，建议卸载后重新安装（控制面板->程序->删除程序）

3.配置环境变量

目的：让当前系统能够识别到Java的运用程序

位置：计算机/此电脑 右键 -> 属性 -> 高级系统设置 -> 高级卡页 -> 环境变量

当前用户变量 和 系统变量 的区别：

在当前用户变量配置的信息，只能在当前用户上使用

在系统变量配置的信息，切换用户后也可以使用

配置信息：JAVA_HOME、Path

注意：

  1. JAVA_HOME一般是没有的，需要新建，Path是有的不需要新建
  2. Path中可能会存在多个路径，多个路径之间使用英文的;作为分隔符
变量名变量值
JAVA_HOMEC:\Program Files\Java\jdk1.9.0_144
Path%JAVA_HOME%\bin

4.环境变量配置是否成功

打开的系统的控制台/cmd/黑窗口/命令行：搜索cmd（按键盘win+r,弹出运行界面，输入“cmd” ）

输入：

​ java -version

​ javac -version

注意：

  1. 如果出现“不是内部或外部命令”，证明配置失败，原因是Java_HOME或Path的路径配置错误
  2. 出现配置失败，修改了路径后，重新启动cmd

三、编写第一个Java程序

1.常用cmd命令

命令描述
cd/退回到根目录
d:跳转到指定D盘符
cd 文件夹名进入到指定文件夹
cls清空cmd命令
dir遍历该文件夹下所有的文件

2.在记事本编写Java源程序

工作空间：编写代码的文件夹（fy2023workspace）

注意：

  1. 老师的项目Day01，学生的项目MyDay01
  2. java源文件的扩展名是Hello.java

3.在cmd命令窗口编译、执行

javac Hello.java – 编译：将.java文件编译成.class文件

java Hello – 运行：运行.class文件

总结：先编译，再运行

4.程序编译执行过程

见java编译运行图.png

四、老师对你们的几点期望

1.按时出勤，不要在人间仙界来回穿梭，更不要四海八荒到处溜达；

2.上课要带元神来，不能只来仙体；

3.老师会把几十年的修为毫无保留滴渡给你，帮你渡劫；

4.自己要努力，争取顺利飞升上神，最起码也要上仙，千万不要应劫；

5.学好这门课，受益三生三世，收获十里桃林；

总结

  1. 软件基础，让同学们有一定的认知
  2. 搭建环境（JDK、JRE、环境变量）
  3. 编写Java代码（工作空间的概念，写代码，编译运行的过程）
