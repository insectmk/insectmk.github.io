增加fail2ban

安装fail2ban
Fail2ban 是一个用于防御暴力攻击的开源工具。
它通过监控系统日志文件，检测恶意行为并自动采取措施来保护 Linux 服务器免受攻击。
Fail2ban 可以分析登录失败的日志记录，并根据预定义的规则和策略来禁止攻击者的 IP 地址。
