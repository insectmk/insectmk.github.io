星露谷-QQ农场说明

星露谷-QQ农场说明

简介

星露谷8人存档，别名《QQ农场》，为了打造一个随时随地能玩儿，不用等房主启动游戏的服务器。

你的劳动成功将会被所有人看见！！！

游戏版本：1.6.15

服务器特性

  • 主持机器人，默认晚上10点睡觉，想要跳过今天，需要在10点睡觉
  • 当服务器只剩主持机器人后，主持机器人会暂停，防止时间流逝

游玩技巧

星露谷物语官方维基、星露谷物语维基-BiliBili游戏

服务器Mod记录

内置模组列表

这些模组都是按照《使用Docker搭建星露谷最新服务端（Steam下载版）》创建服务端后，默认的Mod情况

  • Always On Server(启用)

    如果游戏卡住，玩家可以在聊天框中输入以下指令：

    • !sleep：若当前时间超过设定的就寝时间（默认为晚上10点），此指令将尝试触发睡觉命令。

    • !festival：若当天是节日，此指令将尝试触发前往节日的命令。

    • !event：此指令将尝试触发开启节日主要活动的命令。

    • !leave：此指令将尝试退出节日。

    • !unstick：此指令会将主机在农舍内/外进行传送（通常可突破程序错误，有时需多次传送，尤其在节日场景时见效）。

  • Auto Load Game(启用)

  • Crops Anytime Anywhere(禁用)

  • Friends Forever(禁用)

  • No Fence Decay(禁用)

  • Non Destructive NPCs(禁用)

  • Remote Control(启用)

  • TimeSpeed(禁用)

  • Unlimited Players(启用)

  • 拖拉机(启用)

客户端模组推荐

在安装模组之前，需要安装SMAPI、可视化Mod菜单

通用模组

  • FishingAssistant2–钓鱼助手2
  • Movement Speed-快速移动
  • 自动化
  • 矿洞梯子定位
  • 拖拉机
  • 拖拉机皮肤
  • Mini Bars - Healthbars Mod–怪物血条显示
  • Lookup Anything–查找一切
  • Visible Fish - see catchable fish in the water–可视化鱼

PC模组

  • Lenient Window Resize–调整窗口大小
  • Infinite Zoom–调整游戏大小

移动端模组

  • VirtualKeyboard-虚拟键盘：有些模组需要用到键盘的快捷键，例如：钓鱼，需要用到F5、F6

如何进入

Windows

多人联机 -> 加入 -> 局域网 -> 输入服务器IP -> OK

Android

星露谷是多端互通的，可以到Google Play - Stardew Valley下载安卓版本。

移动端联机

默认移动端是没有开放联机入口的，处于测试阶段，可以参考星露谷物语 (Stardew Valley) 联机指南 | SakuraFrp 帮助文档进行开启。

下面复制一下该网站的文章内容，以防止内容丢失：

目前移动版星露谷物语的联机功能仍可能不稳定，因此作为测试版功能存在，
您需要使用下面的秘籍来开启多人游戏功能。

首先，您应当点击右下角的语言列表，并将游戏切换到英文（English）语言。
目前这个秘籍只能在英文版界面使用，在您开启之后即可切换回中文。

现在我们有英文版的游戏主界面，在这个界面中可以看到招牌上有四个叶子，
在下面图中圈出并以 ↑ ↓ ← → 来表示这四个叶子，
依次点击 ↑ ↑ ↓ ↓ ← → ← →，然后点击右下角的问号按钮，即可解锁多人模式。
